# Advent of Code 2022
Learning Go through Advent of Code.

## Usage
Run test for a specific day:
```
go test ./pkg/day01
```

Run solution for a specific day:
```
go run main.go day01
go run main.go day01 -i input/day01.txt
```

## Usage (compiled)
Compile the module:
```
make
```

Run solution for a specific day:
```
./aoc2022 day01
./aoc2022 day01 -i input/day01.txt
```
